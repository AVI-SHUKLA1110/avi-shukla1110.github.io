# Start Cybersecurity the Minimalist Way — The Only Roadmap You Need 🛣️

Cybersecurity feels overwhelming when you're starting out. Too many tools. Too many paths. Here's a simple, structured roadmap for beginners — focused only on what actually matters, based on my own experience and conversations with other professionals in the industry 👇

## Let's Start

![Perry the Platypus](https://media1.tenor.com/m/0Vi85rk0ISoAAAAd/perry-the-platypus-perry.gif)

### Step 1: Infosec Basics
(If you are new to computer science, I recommend watching this or else directly jump to later parts)

- CompTIA A+ Certification Course
- TryHackMe: Beginner Level - [Link](https://tryhackme.com/hacktivities)
- Security Basics - CompTIA Security+ Certification Course

### Step 2: Networking
If you don't understand networking, cybersec will feel like magic.

Learn:
- How the internet works
- TCP/IP, DNS, HTTP
- IPv4 / IPv6
- Routers, firewalls, Wi-Fi

👉 CompTIA Network+ level knowledge is enough to start.
➕ You can also practice on Cisco Packet Tracer software.

### Step 3: Learn Linux
This is where you'll be spending most of your time.

Resources:
- Linux Essentials for Hackers (YT)
- [explainshell.com](http://explainshell.com) (bookmark this) - this I found out later but was really helpful
- "OverTheWire" - Bandit challenges

### Step 4: Programming (Scripting & Automation)
Either Bash or Python.

**Python:**
- PhD Security - he is an amazing content creator in cybersec and you will learn a lot from his journey: [YouTube](https://youtube.com/watch?v=XWuP5Yf5ILI)
- FreeCodeCamp - Offers tutorials for beginners and this one covers the basics of Python programming: [YouTube](https://youtube.com/watch?v=rfscVS0vtbw)
- Paid but awesome courses - Python101 & Python201 (TCM Security)

Well, if you are looking to go into web security, learning JavaScript would prove to be helpful:
- Academind - an amazing playlist that I personally liked

**For Bash:**
- TryHackMe lesson is a good starter
- Another great resource is this playlist by @HackerSploit

### Step 5: Core Cybersecurity
Now here comes the interesting part - you all have been waiting for 'HACKING'

For intros, these two are my personal favorites:
- Practical Ethical Hacking by @TCMSecurity
- This course by FreeCodeCamp

This will utilize the skills you learnt up till now such as Linux, Python, etc.

🚩 After this point you are ready to start practicing on platforms such as @hackthebox_eu, @tryhackme, @picoctf.

HTB is not so beginner-friendly so you might want to start from TryHackMe -> then move to Hack The Box and PicoCTF and other platforms for hands-on learning.

### Step 6: Choose Your Niche
Cybersecurity in itself is a huge domain so it is practically not feasible to learn everything when you start, and from the knowledge you collected from the above steps you have transitioned beyond a novice and thus it is time for you to explore diverse cybersecurity domains:

For beginners you might want to learn about either of them - application security, network security, penetration testing, SOC, and other blue teaming stuff in general. (Don't forget most of the jobs are in blue teaming)

And for experienced individuals it might be - binary exploitation, malware analysis, cryptography, etc. Identify your interests and strengths to choose a specific niche.

After this point, each one has its own specific resources and roadmaps. Stay curious, dedicated, and keep exploring the dynamic world of cybersecurity.
